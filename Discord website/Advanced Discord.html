Advanced Discord Features and what they do

Discord Nitro

Discord Nitro is some sort of premium thing for discord you can buy to unlock multiple perks. Those perks comes such as:

  • Setting a custom profile in all servers you are in
  • Styling your profile with a custom banner, animated pfp, custom discord tags
  • Nitro exclusive chat stickers
  • Use emojis from a server anywhere
  • Get 2 server boosts to support server + 30% off on all future boosts
  • Discord nitro badge
  • bigger upload size up to 100MB
  • HD video sharing for video calls or screensharing
  • Longer messages! Original character cap: 1000 characters. With nitro = 4000 characters
  • Able to join up to 200 discord serers at the same time

Discord Nitro comes with all of those fabulous perks and discord nitro premium only costs a total of $10 to buy!

Discord Developer Mode

Discord Developer Mode is a setting you can find in the advanced app settings on your discord.
It allows for easier access to things like discord tags and such for easier use of creating and/or using or testing Discord bots!

Server Nitro

Nitro boosting a server allows the server members and owners know that you support them and allow everybody + you in the server access to special perks for each and all different boost levels!
For boosting a discord server, you get a custom role made for you in the server by the name of "Server booster,"
at which the owners of the discord server can set special perks and permissions for that role so you can do things that other people cannot!
When you boost a discord server, it gets you closer to the multiple different tiers of boosts on a server!

  • 2 boosts = Friends tier
  • 7 boosts = Groups tier
  • 14 boosts = Communities tier

Each tier has their own permissions and of course as you guessed, Communities tier has the most perks for everybody on the server!

  1. Friends tier:
    • - Allows you to add Custom stickers and adds more emoji slots
    • - Gives access to everyone on that server better audio and video quality when streaming on that server!
  2. Groups tier:
    • - Everything Friends tier provides
    • - Allows you to upload a custom server banner
    • - Allows members to stream in 1080p and allows the server administrators to create private threads!
  3. Communities tier:
    • - Everything in the friends and groups tier
    • - Allows you to create custom invite links
    • - Allows access to an animated server icon, animated banner, and a custom server invite background!
    • - Allows all members in the server access to 1080p/60fps streaming quality and high quality audio with 100MB uploads!

Server Community

Making your server a community server comes with many perks such as server tools to help out with moderation, running the server, and to grow your server!
Enabling community on your server comes with an entire seperate server settings category to play around with that includes the overview page, Server Insights, Partner Program, Discovery, Membership Screening, and the Welcome Screen!

  1. Overview Page

    2. The Overview page of a discord server allows you to apply:

    • A Rules or Guidelines Channel
    • A Community Updates Channel (notfies server admins of new perks on the server such as threads, etc.)
    • Set the server's primary language
    • Set a server description
    • An option to disable Community

  2. Server Insights page

    3. The Server Insights page of a discord server provides server admins with analytical data about who has joined their server, looked at the rules, welcome, etc.
    If your server has 500+ members in it, you get access to more analytical data from your server insights data such as:

    • Growth & Activation stats
    • Engagement stats (actively chatting members)
    • Audience stats
    • Partner Program

  3. Partner Program

    • Must be a Safe Environment
    • Must have at least 500 members
    • Server must be more than 8 weeks old
    • Server must have active members
    • Moderators on the server must be required to have 2FA enabled on their account

    Some perks that a Partnership on your server is:

    • Unique Branding

      • Custom server url, Server banner, and invite splash

    • Partner-only Perks

      • Get gifted Discord Nitro, and other rewards to host giveaways to your community and allows access to the partners-only discord server

    • Be recognized

      • Gives your server a special badge to stand out frmo the crowd on the discorvery page

  4. Discovery Page

    5. Enabling Server Discovery on a discord server allows your server and an invite to your server to be seen to all members from all over on the Discord Discovery Page.
    Server Discovery comes with the same requirements as a discord partnership but instead of 500 members, you must have 1000 members

  5. Membership Screening Page

    6. Allows all new members to the discord server to be required to read a certain channel like a rules channel and agree to them before they can type or read any content on the server

  6. Welcome Screen Page

    7. When enabled shows a custom list of channels in a server that an administrator recommends to new members to read or check up on and provides a short description of the server!

Discord Hypesquad

HypeSquads are a little nod to Harry Potter as the squads represent different houses to be a part of through a personality quiz! I personally (Michael Porter) am in the House of Brilliance.
There are 3 different houses to be apart of:

  1. The house of Bravery, you like to take risks
  2. The house of Billiance, you are generally pretty smart and think logially
  3. The house of Balance, you take nobody's side on things and try to make things more even and fair

Being apart of a discord HypeSquad provides you with multiple perks such as:

  • Gaining access to a Super Secret Newsletter
  • Having a Hype Profile Badge to display to everyone else
  • Squad challenges to compete against other houses

Being apart of a HypeSquad also allows you to rep HypeSquad events as well! It allows you to register to host or moderate at several real life HypeSquad events.
For repping at a HypeSquad event, discord will send you HypeSquad merch and swag to keep for yourself and to hand out to others during HypeSquad events!

Streamer Mode

Community streamers can enable streamer mode on discord to hide all personal information that may be revealed in normal circumstances. With Streamer Mode on, it will automatically hide all that information!
